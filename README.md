# Dictation Studio

Web app luyện chép chính tả với video YouTube.

## Cấu trúc
- `index.html` — cấu trúc giao diện
- `styles.css` — toàn bộ giao diện và responsive
- `app.js` — logic phát video, kiểm tra đáp án, hint, review, shadowing

## Chạy local
Mở `index.html` trực tiếp hoặc dùng Live Server trong VS Code.

## Ghi chú
Bản hiện tại dùng dữ liệu transcript demo trong `app.js`.
Để dùng video khác, thay `videoId` và mảng `segments` trong `demoLesson`.
